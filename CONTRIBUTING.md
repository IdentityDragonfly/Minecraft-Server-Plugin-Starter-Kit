# Contributing

Thanks for helping improve this Minecraft plugin starter kit.

## How to contribute

1. Fork the repository.
2. Create a feature branch.
3. Make your changes.
4. Test the build with `gradle build`.
5. Open a pull request.

## Good contribution ideas

- Add more commands
- Add more GUI examples
- Add persistent cooldown storage
- Add SQLite/MySQL support
- Improve documentation
- Add tests
