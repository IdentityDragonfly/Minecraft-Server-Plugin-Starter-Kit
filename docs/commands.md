# Commands

| Command | Permission | Description |
|---|---|---|
| `/starter help` | `starter.use` | Shows the help menu. |
| `/starter kit` | `starter.kit` | Gives the starter kit. |
| `/starter menu` | `starter.menu` | Opens the GUI menu. |
| `/starter reload` | `starter.admin` | Reloads the configuration. |

## Aliases

- `/starterkit`
- `/skit`
